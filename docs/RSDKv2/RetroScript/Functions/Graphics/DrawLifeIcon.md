# DrawLifeIcon

Draws the first player's life icon animation at frame `0` at a specified X and Y position in screen-space.

## Parameters
`iXPos`

:   The position of the icon on the horizontal axis in screen-space.

`iYPos`

:   The position of the icon on the vertical axis in screen-space.

## Return Value
None.

## Syntax
```
DrawLifeIcon(int iXPos, int iYPos)
```

## Example
```
DrawLifeIcon(40, 200)
```
