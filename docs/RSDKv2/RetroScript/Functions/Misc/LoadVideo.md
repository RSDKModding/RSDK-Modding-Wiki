# LoadVideo

Loads a video and assigns it to SpriteFrame `0`. The video must be a `.rsv` file.

## Parameters
`Path`

:   The file path of the video file to load, relative to `Data/Sprites/`.

## Return Value
None.

## Syntax
```
LoadVideo(string Name)
```

## Example
```
LoadVideo("Cutscenes/Opening.rsv")
```
