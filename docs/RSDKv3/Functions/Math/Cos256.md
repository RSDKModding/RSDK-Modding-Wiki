# Cos256

## Description
Gets the value from the `cos256` lookup table based on `Angle` and sets it in `Store`.

## Parameters
`Store`

:   The variable to store the result.

`Angle`

:   The value of angle.

## Return Value
Sets `Store` to the result.

## Syntax
```
Cos256(var Store, int Angle)
```

## Example
```
Cos256(TempValue0, Object.Value2)
```
