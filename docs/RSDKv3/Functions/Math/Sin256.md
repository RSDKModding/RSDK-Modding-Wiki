# Sin256

## Description
Gets the value from the `sin256` lookup table based on `Angle` and sets it in `Store`.

## Parameters
`Store`

:   The variable to store the result.

`Angle`

:   The value of angle.

## Return Value
Sets `Store` to the result.

## Syntax
```
Sin256(var Store, int Angle)
```

## Example
```
Sin256(Object.Value3, TempValue0)
```
