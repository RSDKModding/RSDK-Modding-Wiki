# SEGA Classics

## Overview
![Fire TV Game Select](/assets/images/SegaClassics/FTVP-GameSelect.png){ align=right , width="49%" }
SEGA Classics is a now-deleted compilation released in 2018 exclusively for Amazon Fire TV devices. It features a collection of 25 games total. 22 being emulated Sega Genesis/Mega Drive games and the 3 RSDK games at the time, those being Sonic 1, Sonic 2 and Sonic CD.


## Sonic CD (2018)
![RSDKv6 HW Menu](/assets/images/SegaClassics/RSDKv6-HwMenu.png){ align=right , width="49%" }
This compilation has become notorious for including a port to [RSDKv4](/RSDKv4) of Sonic CD.
