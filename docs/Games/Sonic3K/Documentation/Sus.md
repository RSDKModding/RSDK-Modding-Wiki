# Sus

![The Sus animation viewed in RetroED](/assets/images/RetroED/AnimationEditor-Sus.png){ align=right , width="50%" }

Sus is an animation located in the unused animation file `3K_LBZ/ExplodingTrigger.bin` in [Sonic 3 & Knuckles](../README.md). The animation contains one frame, that being the Door Trigger sprite, and is the only animation in the file.

Sus is named after how the Door Trigger resembles the Crewmate characters from the game [Among Us](https://en.wikipedia.org/wiki/Among_Us), with "sus" being a term commonly accosiated with the game. The animation was named by DashPadSPD, who set up the initial game assets for the programmers to implement.

Although the animation and the file it's contained in were both renamed to `Idle` and `DoorTrigger.bin` respectively, the original file was left in the retail build, likely as a result of Sonic Team copying new files when pulling changes from Headcannon's repository without cleaning up old, unused files.
