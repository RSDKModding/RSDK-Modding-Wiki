# Cos

## Description
Gets the value from the `cos512` lookup table based on `angle` and sets it in `store`.

## Parameters
`store`

:   The variable to store the result.

`angle`

:   The value of angle.

## Return Value
Sets `store` to the result.

## Syntax
```
Cos(var store, int angle)
```

## Example
```
Cos(temp0, object.value2)
```
