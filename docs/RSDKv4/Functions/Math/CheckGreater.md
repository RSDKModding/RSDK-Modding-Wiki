# CheckGreater

## Description
Checks if `x` is greater than `y` and stores the result in `checkResult`.

## Parameters
`x`, `y`

:   The values to check.

## Return Value
Sets `checkResult` to the result.

## Syntax
```
CheckGreater(int x, int y)
```

## Example
```
CheckGreater(object.value0, 23)
```
