# SendValue

## Description
Sends a value to the opponent in 2P VS.

## Parameters
`value`

:   The value to send.

`verify`

:   Verifies the object.

## Return Value
None.

## Syntax
```
CallNativeFunction2(SendValue, int value, bool verify)
```

## Example
```
CallNativeFunction2(SendValue, 50, false)
```
