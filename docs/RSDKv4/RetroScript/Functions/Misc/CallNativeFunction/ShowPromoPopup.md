# ShowPromoPopup

Shows a promo popup.

## Parameters
`popupName`

:   The name of the promo popup.

`id`

:   The ID of the promo popup.

## Return Value
None.

## Syntax
```
CallNativeFunction2(NativeFunction, string popupName, int id)
```

## Example
```
CallNativeFunction2(NativeFunction, "My Popup", 0)
```
