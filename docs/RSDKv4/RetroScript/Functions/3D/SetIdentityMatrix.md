# SetIdentityMatrix

## Description
Sets the matrix of `matID` to the identity state.

## Parameters
`matID`

:   The matrix to set the identity. Valid matrices are `MAT_WORLD`, `MAT_VIEW`, and `MAT_TEMP`.

## Return Value
None.

## Syntax
```
SetIdentityMatrix(int matID)
```

## Example
```
SetIdentityMatrix(MAT_WORLD)
```
