# CheckNotEqual

## Description
Checks if `x` and `y` are not equal and stores the result in `checkResult`.

## Parameters
`x`, `y`

:   The values to check.

## Return Value
Sets `checkResult` to the result.

## Syntax
```
CheckNotEqual(int x, int y)
```

## Example
```
CheckNotEqual(object.frame, 3)
```
